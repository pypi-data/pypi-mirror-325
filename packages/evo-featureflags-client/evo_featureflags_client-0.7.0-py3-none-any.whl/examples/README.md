Examples
========

Here you can find examples for HTTP clients with:

- `aiohttp`
- `httpx`
- `requests`

Prerequisites:

- async + http:

  > pip install featureflags-client[httpx]

    or 

  > pip install featureflags-client[aiohttp]

- sync + http:

  > pip install featureflags-client[requests]
